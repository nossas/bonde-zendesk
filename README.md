# Tapioca zendesk

## Installation
```
pip install tapioca-zendesk
```

## Documentation
``` python
from tapioca_zendesk import Zendesk


api = Zendesk(
	api_key='{your-api-id}',
    api_secret='{your-api-secret}',
    access_token='{your-access-token}',
    access_token_secret='{your-access-token-secret}')

```
